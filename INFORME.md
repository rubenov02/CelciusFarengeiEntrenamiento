# Informe de Actividades  
## Repositorio: CelciusFarengeiEntrenamiento

### Introducción

El presente informe documenta el trabajo realizado en el repositorio **CelciusFarengeiEntrenamiento**, cuyo objetivo principal fue la implementación de un modelo de aprendizaje automático sencillo utilizando la librería Scikit-learn para resolver la conversión de temperaturas entre grados Celsius y Fahrenheit. Todo el desarrollo y análisis se realizó en un entorno **Jupyter Notebook**, facilitando la presentación interactiva de los resultados y el código fuente.

### Descripción de las Actividades

1. **Planteamiento del Problema**
   - Se formuló el problema de conversión de temperaturas como una tarea de regresión lineal supervisada, donde la variable independiente es la temperatura en grados Celsius y la variable dependiente es la temperatura en grados Fahrenheit.

2. **Preparación del Conjunto de Datos**
   - Se generaron o recopilaron pares de datos de temperaturas equivalentes en Celsius y Fahrenheit, asegurando que el conjunto fuera representativo para entrenar adecuadamente el modelo.

3. **Implementación de la Solución**
   - Utilizando Python y la biblioteca Scikit-learn, se instanció y entrenó un modelo de regresión lineal para ajustarse a la relación matemática entre ambas escalas de temperaturas.
   - Se realizó la partición de los datos en conjuntos de entrenamiento y prueba para validar el desempeño del modelo.

4. **Evaluación y Resultados**
   - Se evaluó el modelo mediante métricas apropiadas, comprobando que los parámetros estimados se aproximan a la fórmula teórica de conversión.
   - Se visualizó la relación y los resultados a través de gráficos generados en el propio notebook, facilitando la interpretación y comprensión del proceso.

5. **Conclusiones**
   - El ejercicio permitió consolidar conceptos básicos de regresión lineal y su implementación en Python para problemas sencillos de conversión numérica.
   - La experiencia refuerza la importancia de las herramientas interactivas (como Jupyter Notebook) para el desarrollo, documentación y presentación de experimentos de aprendizaje automático en el ámbito académico.

### Tecnologías Utilizadas
- Python
- Jupyter Notebook
- Scikit-learn
- Matplotlib (para visualización, si aplica)

### Consideraciones Finales

El repositorio constituye una guía introductoria útil, tanto para estudiantes que inician en el campo del aprendizaje automático, como para quienes deseen ver una aplicación práctica y didáctica del ajuste de modelos lineales a problemas matemáticos clásicos.
